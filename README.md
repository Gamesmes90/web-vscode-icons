# web-vscode-icons
Vscode icons for github, gitlab, sourceforge and gitea

Inspired by [github-vscode-icons](github-vscode-icons), this extensions adds some bug fixes and new features.

## Features

- [x] GitHub
- [x] GitHub Directory
- [x] GitLab
- [x] GitLab Directory
- [x] SourceForge
- [x] Gitea
- [x] Gitea Directory
- [ ] Options menu
- [ ] Custom entries
